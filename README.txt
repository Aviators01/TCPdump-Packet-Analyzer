Description: This program displays packets details that are captured from TCPdump. Program abilities include display of IP packets,
ARP packets, and port numbers of senders/receiver ip addresses.

Limitations: Will not process byte code files. They must be text. Limited error checking if wrong arguments entered. Unable to
display IP6 packets or any other type of packets including ICMP.

Instructions: In the Ubuntu terminal window:
		1. Navigate to folder where program and data file stored
		2. Run by typing: python3 TCPAnalyzer.py
